# 🏠 Линейная регрессия для предсказания цен на жилье

## 📋 Описание проекта
Проект по машинному обучению для предсказания рыночной стоимости жилой недвижимости на основе различных характеристик.

## 🎯 Цель работы
Построить модель линейной регрессии, которая предсказывает цену дома на основе:
- Количества спален и ванных комнат
- Жилой и общей площади
- Года постройки и реновации
- Оценки качества и состояния
- Географического расположения (широта/долгота)

## 📊 Данные
- **Обучающая выборка**: 15,129 записей
- **Тестовая выборка**: 6,484 записи
- **Признаки**: 16 характеристик недвижимости

## 🛠 Используемые технологии
- Python 3.11
- Pandas, NumPy - обработка данных
- Scikit-learn - машинное обучение
- Matplotlib, Seaborn - визуализация
- OpenPyXL - работа с Excel файлами

## 📈 Модели
| Модель | R² (валидация) | RMSE | MAE |
|--------|----------------|------|-----|
| Linear Regression | 0.6947 | $189,231 | $121,912 |
| Ridge Regression | 0.6948 | $189,209 | $121,906 |
| Lasso Regression | 0.6947 | $189,231 | $121,912 |

**Лучшая модель**: Ridge Regression

## 🔍 Ключевые выводы
- Жилая площадь (sqft_living) - самый важный фактор
- Оценка качества (grade) сильно влияет на цену
- Модель объясняет ~70% вариации цен
